# Stone-Paper-Scissor-Game

✊✋✌️ Stone Paper Scissor Game
A simple and fun web-based **Stone Paper Scissor** game where you play against the computer.


 🚀 Features
* Play against computer 🤖
* Random computer moves
* Instant result display
* Simple and interactive UI


🛠️ Tech Stack
* HTML
* CSS
* JavaScript


▶️ Run Project
Open `index.html` in your browser.



🎮 How to Play
* Choose **Stone, Paper, or Scissor**
* Computer selects randomly
* Winner is decided instantly:

  * Stone beats Scissor
  * Scissor beats Paper
  * Paper beats Stone
